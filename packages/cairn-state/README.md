# @isaacriehm/cairn-state

Lightweight state management package for Cairn.

- Ground state schemas (Zod)
- Path resolution for `.cairn/`
- Ledger and task readers (cached)
- Low-level I/O primitives
- Decoupled logger interface

Internal use by `@isaacriehm/cairn-core` and `@isaacriehm/cairn-lens`.
